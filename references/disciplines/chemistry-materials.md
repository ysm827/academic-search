# Chemistry and Materials Discipline Profile

## Use When

Use for chemistry, chemical engineering, materials science, catalysis, polymers, batteries, nanomaterials, and molecular discovery.

## Platform Route

| Need | Preferred Sources |
|------|-------------------|
| Cross-disciplinary metadata | Crossref, OpenAlex |
| Preprints | ChemRxiv, arXiv where relevant |
| Publisher pages | ACS, RSC, Wiley, Springer, Elsevier |
| Open access status | Unpaywall |

## Query Expansion

- Include compound synonyms, material formulas, reaction names, and CAS numbers when known.
- Separate experimental synthesis, characterization, computation, and review papers.
- Add method names such as DFT, XRD, SEM, TEM, NMR, GC-MS when relevant.

## Ranking

1. Exact material/compound/reaction match.
2. Review papers for broad surveys; experimental papers for protocols and measured properties.
3. Journal/source quality and reproducibility signals.
4. Open data and supplementary information availability.

## Output Fields

Include DOI, publication type, material or compound keywords, method keywords, open_access_status, full_text_status, and supplementary/data availability when available.
