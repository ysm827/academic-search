# Economics and Social Science Discipline Profile

## Use When

Use for economics, management, sociology, psychology, political science, education, communication, and policy research.

## Platform Route

| Need | Preferred Sources |
|------|-------------------|
| Economics working papers | RePEc, NBER, SSRN |
| Psychology / social science preprints | PsyArXiv, OSF |
| Cross-disciplinary metadata | OpenAlex, Crossref |
| Citation counts | Google Scholar, OpenAlex |
| Open access status | Unpaywall |

## Query Expansion

- Use JEL codes for economics when possible.
- Include method terms: RCT, difference-in-differences, regression discontinuity, survey, interview, ethnography.
- Include geography, population, policy, and time-period terms when relevant.

## Ranking

1. Match method and dataset to the user's question.
2. Prefer peer-reviewed papers for settled claims; include working papers for frontier policy/economics topics.
3. Consider institution/source quality for working papers.
4. Use citation counts cautiously because subfields vary widely.

## Output Fields

Include DOI, working paper source, JEL codes, method type, sample/population, geography, publication status, open_access_status, and full_text_status.
