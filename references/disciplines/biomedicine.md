# Biomedicine Discipline Profile

## Use When

Use for medicine, clinical research, biology, genomics, neuroscience, public health, pharmacology, epidemiology, and life-science literature.

## Platform Route

| Need | Preferred Sources |
|------|-------------------|
| Biomedical search | PubMed |
| Open full text | PubMed Central, Europe PMC, Unpaywall |
| Preprints | bioRxiv, medRxiv |
| Clinical trials | ClinicalTrials.gov |
| DOI / publisher metadata | Crossref |
| Cross-disciplinary citations | OpenAlex, Semantic Scholar |

## Query Expansion

- Use MeSH terms when available.
- Include disease synonyms, drug generic names, and common abbreviations.
- For clinical questions, expand PICO elements: population, intervention, comparator, outcome.
- Keep preprints separate from peer-reviewed results when clinical reliability matters.

## Ranking

Read `references/rankings/biomed-evidence-ranking.md` before prioritizing papers. Prefer systematic reviews, meta-analyses, randomized controlled trials, and clinical guidelines over narrative reviews or case reports when the user asks for evidence.

## Output Fields

Include PMID, PMCID, DOI, publication type, study type, sample size, population, MeSH terms, open access status, and full_text_status when available.

## Safety Boundary

Do not turn search results into medical advice. Summarize evidence and uncertainty, and recommend consulting qualified professionals for clinical decisions.
