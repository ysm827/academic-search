# Physics and Mathematics Discipline Profile

## Use When

Use for physics, astronomy, high-energy physics, mathematics, statistics, applied mathematics, and theoretical work.

## Platform Route

| Need | Preferred Sources |
|------|-------------------|
| Preprints | arXiv categories |
| Astronomy / astrophysics | NASA ADS |
| High-energy physics | INSPIRE HEP |
| Math classification | MSC where available |
| DOI / journal metadata | Crossref |
| Cross-disciplinary citations | OpenAlex, Semantic Scholar |

## Query Expansion

- Use arXiv categories such as `hep-th`, `astro-ph`, `math.AG`, `stat.ML`.
- Include theorem names, model names, experiment names, and common notation variants.
- For mathematics, include MSC terms or problem names when known.

## Ranking

1. Match to exact subfield/category.
2. Seminal papers and surveys when building background.
3. Recent preprints when tracking frontier work.
4. Citations and venue only after subfield relevance is established.

## Output Fields

Include arXiv ID, DOI, category, venue, abstract, citation count, and open PDF link.
